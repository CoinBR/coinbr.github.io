Description
Main Description

Quando uma História de Usuário é mal compreendida, talvez texto não esteja sendo a maneira mais eficiente daquele usuário comunicar sua necessidade.
Gravar um aúdio ou vídeo, explicando a funcionalidade, certamente facilitará a compreenção por parte dos Desenvolvedores.

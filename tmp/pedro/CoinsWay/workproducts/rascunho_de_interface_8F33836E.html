Purpose

Mostrar aos Desenvolvedores como deverá ficar visualmente uma funcionalidade no sistema

Relationships

Output From:
Description
Main Description

A interface projetada por um Desenvolvedor para entregar uma funcionalidade, pode, muitas vezes, ser diferente do que o Usuário espera.
Algo que faz sentido para um desenvolvedor, pode ser contra-intuitivo, ou até burocrático para o Usuário.
Desenhar um rascunho da interface desejada, é uma boa solução para esses problemas.

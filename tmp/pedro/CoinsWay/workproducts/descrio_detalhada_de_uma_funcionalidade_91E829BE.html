Artifact: Descrição Detalhada de uma Funcionalidade
Purpose
Fazer os Desenvolvedores Compreenderem com Clareza uma funcionalidade/necessidade que foi mal entendida por eles
Relationships
TasksInput To:
  • Detalhar uma Funcionalidade mal entendida
  • Implementar uma Funcionalidade
Output From:
Description
Main DescriptionSe mesmo com a reunião presencial os Desenvolvedores não entenderam a funcionalidade solicitada em uma História de Usuário, então possivelmente uma História de Usuário não seja a melhor maneira de descrever tal funcionalidade.

Um Caso de Uso, ou qualquer outro tipo de texto detalhado poderão ajudar os Desenvolvedores a Compreenderem a funcionalidade.

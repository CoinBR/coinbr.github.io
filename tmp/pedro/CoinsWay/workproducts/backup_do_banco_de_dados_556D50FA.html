Artifact: Backup do Banco de Dados
Purpose

* Garantir a integridade dos dados do sistema
* Permitir teste em Produção de uma funcionalidade recém adicionada

Relationships
RolesResponsible: Modified By:
  • Desenvolvedores
TasksInput To:
  • Restaurar Backup da Última Versão de Produção
  • Restaurar Backup do Banco feito Antes dos Testes
Output From:
  • Backup da Versão de Produção
  • Backup do Banco para poder fazer Testes em Produção
Description
Main Description

Backups do Banco de Dados serão feitos antes e depois de subir uma nova funcionalidade para Produção

O backup feito antes será restaurado caso a nova funcionalidade não passe nos testes. Com isso o sistema voltará a funcionar, e os dados terão sido preservados.

O backup feito depois será restaurado caso a nova funcionadade passe nos testes. Com isso o lixo gerado e os dados apagados durante o teste serão desconsiderados.

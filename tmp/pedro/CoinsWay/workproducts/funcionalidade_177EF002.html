Main Description

* Pensamos em um Software como um conjunto de Funcionalidades reunidas em um só lugar.
* Isso ajuda a criar um Sistema incrementalmente.
* Cada necessidade do usuário será atendida por uma mais funcionalidades do Sistema
* Os usuários já serão ajudados pelo sistema, mesmo que ele ainda não atenda todas suas funcionalidades
* A medida que mais funcionalidades vão sendo entregues, os usuários precisarão sair cada vez menos do ambiente do sistema para atender suas necessidades

Task: Backup da Versão de Produção
Purpose
* Garantir a integridade dos dados e processos da Empresa
* Manter o sistema funcionando
Relationships
RolesPrimary Performer:
  • Desenvolvedores
Additional Performers:
InputsMandatory:
  • Funcionalidade
    		Optional:
    • None
    Outputs
    • Backup do Banco de Dados
    • Backup do Código em Produção
      Main Description

      Por mais testada que esteja uma funcionalidade, ela pode gerar problemas quando subir para produção.

      Tal funcionalidade pode não só tirar o sistema do ar, como também comprometer dados que estão no banco de Dados

      Por isso, antes de subir qualquer nova funcionalidade, os Desenvolvedores farão um backup do código, assets e do banco de dados que estão em produção.

Task: Backup do Banco para poder fazer Testes em Produção
Purpose

Permitir que a nova funcionalidade seja testada no sistema em Produção

Relationships
RolesPrimary Performer:
  • Desenvolvedores
Additional Performers:
InputsMandatory:
  • Funcionalidade
    		Optional:
    • None
    Outputs
    • Backup do Banco de Dados
      Main Description

      Com o Banco em Produção já adaptado para a nova funcionalidade, deve ser feito um backup dele.
      Isso permitirá que se faça todo tipo de teste, sem se preocupar com a integridade dos dados já presentes na empresa.

      Se a nova funcionalidade passar em todos os testes feitos em produção, o backup feito nessa tarefa será restaurado, e a função já poderá ser usada em produção.

      Caso a nova funcionalidade não passe nos testes, o backup feito na tarefa "Backup da Versão de Produção" será restaurado.
